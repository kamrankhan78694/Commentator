üí¨ Common Questions

What is Commentator?

Commentator is an open-source universal web commenting system that allows users to leave and view comments on any website, even those that don't have native commenting systems.

How does decentralization work?

We're building toward storing comments on IPFS/Arweave, using wallet-based authentication, and implementing community-driven moderation to ensure no single entity can control or censor the system.

How do I install Commentator?

Currently, you can clone the repository and open index.html in your browser. Browser extensions are coming soon for seamless integration with any website.

Do I need to know how to code to contribute?

Not necessarily! You can contribute by testing, reporting bugs, improving documentation, suggesting features, or spreading the word about the project.

How do you prevent spam and abuse?

We're implementing community moderation with upvote/downvote systems, reputation-based filtering, and exploring crypto-economic incentives to maintain quality.

Which browsers are supported?

Commentator works on all modern browsers including Chrome, Firefox, Safari, and Edge. We use standard web technologies for maximum compatibility.

Is Commentator free to use?

Yes! Commentator is completely free and open-source under the MIT license. It will always remain free for users and open for contributions.

What's the development roadmap?

Current priorities include browser extensions, decentralized storage integration, user authentication systems, and community moderation features. Check our roadmap page for detailed plans.

üîß Technical Questions

How is data stored?

Currently using local storage for development. Future versions will use IPFS and Arweave for decentralized, permanent storage that can't be censored or removed.

How does authentication work?

Future versions will support Web3 wallet authentication (MetaMask, WalletConnect), ENS domains, and Ceramic Network for decentralized identity management.

What about performance?

Commentator is designed to be lightweight and fast. Comments load asynchronously and the interface uses modern web standards for optimal performance.

Does it work on mobile?

Yes! The interface is fully responsive and works on all devices. Mobile apps are planned for future releases to enhance the mobile experience.

üîí Privacy & Security

What data do you collect?

Currently minimal data collection for development purposes. Future versions will be designed for maximum privacy with optional user data stored only on decentralized networks.

How secure are comments?

Comments will be cryptographically signed for authenticity and stored on tamper-proof networks. Users will have full control over their data and identity.

Who controls my data?

You do! With decentralized storage and Web3 identity, you maintain full ownership and control of your comments and personal data.
