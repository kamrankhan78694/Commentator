We're planning a comprehensive JavaScript SDK for easy integration:

Example Usage

// Initialize Commentator
const commentator = new Commentator({
  apiKey: 'your-api-key',
  site: window.location.hostname
});

// Load comments for current page
commentator.loadComments({
  url: window.location.href,
  container: '#comments-container'
});

// Post a new comment
commentator.postComment({
  content: 'Great article!',
  author: 'user123',
  url: window.location.href
});
