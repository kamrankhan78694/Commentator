Simple Embed (Coming Soon)

Add this snippet to any HTML page:

<div id="commentator-comments"></div>
<script src="https://commentator.app/embed.js"></script>
<script>
  Commentator.init({
    container: '#commentator-comments',
    url: window.location.href
  });
</script>

NPM Package (Planned)

Install via npm for React, Vue, or other frameworks:

npm install @commentator/react

import { CommentatorWidget } from '@commentator/react';

<CommentatorWidget url={window.location.href} />
