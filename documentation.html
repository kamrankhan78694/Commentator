Commentator is an open-source project designed to improve transparency on the web by enabling community-driven feedback. It allows users to comment on any website‚Äîeven those that disable native comments or review systems‚Äîhelping others make informed decisions and avoid scams.

The project serves as both a practical tool for web transparency and a foundation for building decentralized communication protocols that serve the public good.

üéØ Project Goals: Decentralization

Our long-term vision is to make Commentator a fully decentralized public commenting system that cannot be controlled or censored by any single entity.

Key Objectives:

üóÇ Decentralized Storage

Store comment data on IPFS or Arweave using tools like web3.storage or Bundlr, ensuring content permanence and censorship resistance.

üÜî Decentralized Identity (DID)

Use Ethereum wallets (e.g., MetaMask) or identity protocols like ENS and Ceramic to allow users to sign comments without centralized accounts.

üó≥Ô∏è Community Moderation

Implement upvote/downvote systems, and explore DAO-style moderation using token-based or reputation-based voting.

üîê Content Integrity

Let users cryptographically sign their comments for authenticity and verifiability.

üí° Open Protocol

Create APIs and standards that can be used by other platforms to fetch or display Commentator data transparently.

This decentralized foundation ensures Commentator is not just a product‚Äîbut a protocol that serves the public good.

‚ùì Frequently Asked Questions

What makes Commentator different from other commenting systems? +

Commentator is designed to be universal and decentralized. Unlike traditional commenting systems that are controlled by website owners, Commentator allows users to comment on ANY website, even those that don't have native commenting systems. Our long-term goal is full decentralization using blockchain and IPFS technologies.

How do I get started with development? +

The easiest way is to clone the repository and open index.html in your browser. No build process is required for basic development. The project uses vanilla HTML, CSS, and JavaScript, making it accessible to developers at all levels.

Is there a browser extension available? +

The browser extension is currently in development. You can track progress on our GitHub repository and contribute to its development. The current web interface serves as a proof of concept and testing ground for the core functionality.

How will decentralization work? +

We plan to implement decentralization through multiple layers: IPFS/Arweave for storage, Ethereum wallets for identity, cryptographic signatures for authenticity, and DAO-style governance for moderation. This ensures no single entity can control or censor the commenting system.

What's the current project status? +

The project is currently in early development with a working web interface and demo functionality. Core features like comment storage, user authentication, and browser extension are in active development. Community contributions are welcome and essential for rapid progress.

How can I contribute without coding? +

Non-coding contributions are valuable! You can help with documentation, design, testing, user experience feedback, spreading awareness, or participating in community discussions. Every contribution helps build a better product for everyone.
